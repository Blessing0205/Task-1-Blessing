## Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a sales dataset using Microsoft Excel. The objective was to clean the data, identify inconsistencies, calculate descriptive statistics, analyze trends, detect outliers, and generate meaningful insights from the dataset.
## Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
GitHub
## Data Cleaning Process
Identified and handled missing values.
Replaced missing CouponCode entries with "No Coupon".
Checked for duplicate records.
Corrected data formats for dates, quantities, and prices.
Ensured data consistency and accuracy.
## Exploratory Data Analysis
Calculated basic statistics (Mean, Median, and Count).
Created Pivot Tables to summarize data.
Generated trend charts to visualize sales performance over time.
Identified potential outliers in transaction values.
Examined customer purchasing patterns.
## Key Findings
309 missing values were identified and handled.
No duplicate records were found.
Sales trends varied across different periods.
A few transactions showed unusually high values, indicating possible outliers or bulk purchases.
The dataset was successfully cleaned and prepared for further analysis.
